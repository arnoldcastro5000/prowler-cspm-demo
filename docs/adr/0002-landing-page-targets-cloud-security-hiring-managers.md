# Landing page targets cloud security hiring managers, not general audiences

The root route (`/`) is a standalone page designed for a hiring manager with a cloud security background — not a general introduction to CSPM. It assumes the reader knows the domain and leads with credibility signals: real infrastructure scanned, not mocked data; Terraform-provisioned resources; credentials in GCP Secret Manager; full pipeline from IaC to Cloud Run to Cloudflare.

The alternative (a tutorial-style explainer defining CSPM from scratch) would waste the attention of the target audience and dilute the portfolio signal. The landing page includes a Mermaid infrastructure diagram, tech stack table, and direct CTAs to the Before/After snapshots and GitHub repo.
